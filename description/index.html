説明

ゲーム紹介

かわいいキリンたんを眺めるゲームです。

操作方法

マウス操作、タップ操作に対応しています。

  • 左右ドラッグ　視点の向きの変更
  • 上下ドラッグ　拡大縮小
  • クリック（タップ）　鳴く
  • 長押し（マルチプレイ時のみ）　ゲーム情報表示

二次利用について

  • ゲーム自体の再配布は禁止です。
  • ゲーム実況、及びそれに類する利用については、常識的な範囲でご自由にどうぞ。
  • なお、このゲームではゲーム作者以外の権利物も使用しております。二次利用の際は各素材の権利もご確認ください。
